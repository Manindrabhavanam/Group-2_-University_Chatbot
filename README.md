AI-Powered University Student Support Chatbot

Description
This chatbot helps students obtain information about admissions, scholarships, and student services.

Technologies Used
- Python
- Gradio
- Hugging Face
- FAISS

Installation

pip install -r requirements.txt

Run

python app.py

Features
- Student FAQ support
- Conversational chatbot
- University information assistance

Hugging Face Model Integration

The chatbot uses the Hugging Face Transformers library with the DistilGPT2 language model. The model generates responses to user queries through a Gradio-based interface.
